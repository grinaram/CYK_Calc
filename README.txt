func.py - реализовывает алгоритм анализа грамматики CYK - The Cocke-Younger-Kasami
https://en.wikipedia.org/wiki/CYK_algorithm

calc.py - калькулятор
матрицы и правила находятся в отдельных файлах

анализ и вычисления в обоих файлах производятся с помощью функции alg, которой на вход нужно подать выражение

для func.py

alg("a+a*(a*a)")

для calc.py

#alg("(101*11)*10+11*(1001+101)+1")

#alg("1010101010")

#alg("10*10+10*10+10*10")

или другие выражения

если на вход будет подано неверное выражение, выведется сообщение "sentence is incorrect"
